# Prismatic Enterprises

An indie operator's portfolio.

I'm Christian. 18 years in IT security and operations, most recently from MSP-land. After getting laid off in March 2026, I started productizing the AI-native helpdesk tooling I'd built, plus a few other things that had been sitting on my drive waiting for time.

Prismatic Enterprises LLC is the parent company for the consulting work and the products. Some live here on GitHub. Some live elsewhere.

## What's here

**[prospect-miner](https://github.com/prismatic-enterprises/prospect-miner)**: Find warm prospects on Reddit by mining declared-intent signals. Configurable ICP scoring, Notion or markdown output. Built for our own customer development; useful for anyone doing the same.

**[outreach-helper](https://github.com/prismatic-enterprises/outreach-helper)**: CLI for managing customer development conversation threads. Pulls history from your tracker, generates next-line drafts in your voice, logs sent messages back. Pairs naturally with prospect-miner.

More tools shipping over the coming weeks: PowerShell M365 tenant audits, helpdesk automation reference architectures, and a few other things from the consulting bench.
